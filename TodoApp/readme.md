# 💻 Real-Time Todo Management System

A professional Todo Management System built using **Node.js**, **Express.js**, **EJS**, and **Bootstrap**.

This project allows users to:

- Add Tasks
- Edit Tasks
- Delete Tasks
- Change Task Status
- Track Task Progress

---

✅ Professional Dashboard UI

# ⚙️ Installation

## Step 1: Clone Project

```bash
git clone <repository-link>
```

---

## Step 2: Open Folder

```bash
cd TodoApp
```

---

## Step 3: Install Dependencies

```bash
npm install
```

---

## Step 4: Run Server

```bash
node app.js
```

---

# 🌐 Run Project

Open browser:

```plaintext
http://localhost:8000
```

# 📌 Core Modules

## 1. Dashboard Module

- Displays task summary
- Shows total, pending, and completed tasks

---

## 2. Task Creation Module

Users can:
- Add task title
- Add description
- Select priority

---

## 3. Task Listing Module

Displays:
- Task ID
- Title
- Description
- Priority
- Status
- Actions

---

## 4. Task Update Module

Users can:
- Edit task details
- Update task information

---

## 5. Task Delete Module

Users can remove tasks permanently.

---

## 6. Status Management Module

Task status flow:

```plaintext
Pending → In Progress → Completed
```
