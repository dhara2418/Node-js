# Node.js Server Project

A simple Node.js server project using `app.js` as the main entry point.
---

## 🚀 Requirements

Before running the project, install:

- Node.js
- npm

Check versions:

```bash
node -v
npm -v
```

---

## ▶️ Run the Project

Start the server:

```bash
node app.js
```
---

## ⚙️ Add Start Script

Inside `package.json` add:

```json
"scripts": {
  "start": "node app.js"
}
```

## 🌐 Open in Browser

```bash
http://localhost:000
```
